# Todo App
## Built with html,css,javascript.

## Features
-Add tasks
-Remove tasks
-Mark tasks as done
-Saves data in local storage


## How it works
This app works by getting the values stored in the allTodos array and for each element stored (by submitting a value with add button that works with addeventlistener) it gives
a nice looking row with the checkmark text submitted and the delete button.It saves in local storage so even if you refresh the page it wont remove the checkmark or the tasks the user submitted.
